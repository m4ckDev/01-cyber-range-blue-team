# Glossary

| Term | Meaning |
|---|---|
| Alert | A detection result that needs review |
| Authentication Log | Log source containing login attempts and account activity |
| Brute Force | Repeated authentication attempts that may indicate guessing or automation |
| Detection | Logic used to identify suspicious or important events |
| Evidence | Log lines or records supporting an analyst conclusion |
| False Positive | Alert that looks suspicious but is not actually malicious |
| File Integrity Monitoring | Tracking file changes, especially to sensitive paths |
| Indicator | Observable evidence such as an IP, username, process, file path, or URL |
| Severity | Estimated business or technical impact |
| Confidence | How strongly the evidence supports the conclusion |
| Timeline | Ordered list of relevant events |
| Triage | Reviewing an alert to decide priority and next action |
